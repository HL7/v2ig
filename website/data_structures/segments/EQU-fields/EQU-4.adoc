== EQU.4 - EQU-4 (CWE)

[segment-definition]
This field identifies the current state of control associated with the equipment. An equipment can either work autonomously ('Local' control state) or it can be controlled by another system, e.g., LAS computer ('Remote' control state). Refer to {{v2-table:0366:Local/Remote Control State}} in Chapter 2C, Code Tables, for valid values.
