== OBX.27 - Root Cause (CWE)

[segment-definition]
This element contains the reason code indicating the root cause for the reissue of a previously released lab report. This element is used in conjunction with _OBX-11 Observation Result Status_ to define the root cause for a reissued laboratory result in the case of a corrected, amended, appended, or revised result. For example, if the laboratory result was reissued due to an equipment malfunction.
