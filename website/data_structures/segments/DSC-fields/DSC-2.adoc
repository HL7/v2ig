== DSC.2 - Continuation Style (ID)

[segment-definition]
Indicates whether this is a fragmented message (see Section _2.9.2_, "_Continuation messages and segments_"), or if it is part of an interactive continuation message (see Section 5.6.3, "Interactive continuation of response messages").

[segment-comment]
Refer to {{v2-table:0398:Continuation Style Code}} in Chapter 2C, Code Tables, for valid values.
