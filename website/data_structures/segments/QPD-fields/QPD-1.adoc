== QPD.1 - Message Query Name (CWE)

[segment-definition]
This field contains the name of the query. These names are assigned by the function-specific chapters of this specification. It is one to one with the Query Profile for this query name, and it is in fact an identifier for that Query Profile. Site-specific query names begin with the letter "Z." Refer to {{v2-table:0471:Query name}} in Chapter 2C, Code Tables, for suggested values.
