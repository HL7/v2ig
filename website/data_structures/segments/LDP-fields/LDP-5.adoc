== LDP.5 - Valid Patient Classes (CWE)

[segment-definition]
This field contains the patient types that are allowed to be assigned to this bed. For example, Inpatient, Outpatient, Series, Clinic, ER, Ambulatory, Observation, etc. These values should be the same set of values as those used for _PV1-2 - Patient Class_. Refer to {{v2-table:0004:Patient Class}} in Chapter 2C, Code Tables, for suggested values.
