== RXD.17 - Actual Strength Unit (CWE)

[segment-definition]
Use when _RXD-2-Dispense/Give Code_ does not specify the strength. This is the unit of the strength, of a single dosage unit of the dispensed product, used in combination with _RXD-16-actual strength_. Refer to Table 0707 - Actual Strength Unit in Chapter 2C for valid values.

[segment-comment]
*Note:* These units can be a "compound quantity;" i.e., the units may express a quantity per unit of time. For example, micrograms per hour (micg/h) is an acceptable value. These compound units are contained in the ISO+ table. See Chapter 7 for full definition of ISO+ units.
