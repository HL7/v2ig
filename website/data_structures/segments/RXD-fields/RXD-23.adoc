== RXD.23 - Dispense Package Size Unit (CWE)

[segment-definition]
This field contains the units in which _RXE-28-dispense package size_ is denominated. The advertised number of units in the manufacturer's package, i.e., the package as it comes from the supplier. Refer to Table 0709 - Dispense Package Size Unit in Chapter 2C for valid values.
