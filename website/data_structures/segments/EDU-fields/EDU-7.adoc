== EDU.7 - School Type Code (CWE)

[segment-definition]
This field contains the categorization of the academic institution that granted the degree to the Staff Member. HL7 suggests using values in {{v2-table:0402:School Type}}.

[segment-comment]
Refer to {{v2-table:0402:School Type}} in Chapter 2C, Code Tables, for valid values. This table contains values for graduate, medical etc.
