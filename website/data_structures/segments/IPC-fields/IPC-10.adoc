== IPC.10 - Action Code (ID)

[segment-definition]
This field reveals the intent of the message. Refer to {{v2-table:0206:Segment Action Code}} for valid values.

[segment-comment]
The action code can only be used when the combination of IPC-1, IPC-2, IPC-3, and IPC-4 represents a unique identifier according to Chapter 2, Section 2.10.4.2.
