== LCC.2 - Location Department (CWE)

[segment-definition]
This field contains the institution's department to which this location belongs, or its cost center. It may match the value in its preceding LDP (_LDP-2 - Location Department_ or _LDP-12 - Location Cost Center_. Refer to {{v2-table:0264:Location Department}} in Chapter 2C, Code Tables, for suggested values.
