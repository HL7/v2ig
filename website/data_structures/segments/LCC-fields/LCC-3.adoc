== LCC.3 - Accommodation Type (CWE)

[segment-definition]
This field contains the financial accommodation type of the bed or room which implies the rate to be used when occupied by a patient under specific medical conditions, which determines how it is billed. Not the same as specialty type. Used for general ledger categories. Specialty type is a physical accommodation type, whereas this field is a financial accommodation type. Repeating coded value. Refer to {{v2-table:0129:Accommodation Code}} in Chapter 2C, Code Tables, for suggested values.
