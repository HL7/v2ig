== BUI.12 - BUI-12 (CNE)

[segment-definition]
The unit of measure of the transport temperature range. Concepts are pulled from the UCUM code system (_http://www.unitsofmeasure.org[www.unitsofmeasure.org])._ Refer to {{v2-table:0931:Temperature Units}} in Chapter 2C, Code Tables, for valid entries.
