== IVT.9 - Issue Packaging (CWE)

[segment-definition]
This field contains the primary issue packaging unit by which the inventory supply item can be issued to departments or other locations by the corresponding inventory location specified in IVT-2. See {{v2-table:0818:Package}} as described in _PKG-2 Packaging Units_, and presented in Chapter 2C, Code Tables, for suggested values.
