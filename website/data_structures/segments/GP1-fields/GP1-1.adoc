== GP1.1 - Type of Bill Code (CWE)

[segment-definition]
This field is the same as UB92 Form Locator 4 "Type of Bill". Refer to {{v2-table:0455:Type of Bill Code}} in Chapter 2C, Code Tables, for suggested values. Refer to a UB specification for additional information. This field is defined by CMS or other regulatory agencies. It is a code indicating the specific type of bill with digit 1 showing type of facility, digit 2 showing bill classification, and digit 3 showing frequency.
