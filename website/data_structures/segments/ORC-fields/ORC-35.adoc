== ORC.35 - Action Code (ID)

[segment-definition]
This field reveals the intent of the message. Refer to {{v2-table:0206:Segment Action Code}} for valid values.

[segment-comment]
The action code can only be used when either ORC-2 and/or ORC 3 is valued with a unique identifier in accordance with Chapter 2, Section 2.10.4.2.
