== ORC.26 - Advanced Beneficiary Notice Override Reason (CWE)

[segment-definition]
This field contains the reason why the patient did not sign an Advanced Beneficiary Notice. The reason may be coded or it may be a free text entry. Refer to {{v2-table:0552:Advanced beneficiary notice override reason}} in Chapter 2C, Code Tables.

[segment-comment]
Condition: This field is required if the value of ORC-20 Advanced Beneficiary Notice Code indicates that the notice was not signed. For example, additional qualifying or explanatory information would be justified if ORC-20 was populated with the values "3" or "4" in {{v2-table:0339:Advanced Beneficiary Notice Code}}, or similar values in related external code tables.
