== ORC.5 - Order Status (ID)

[segment-definition]
This field specifies the status of an order. Refer to {{v2-table:0038:Order status}} in Chapter 2C, Code Tables, for valid entries. The purpose of this field is to report the status of an order either upon request (solicited), or when the status changes (unsolicited). It does not initiate action. It is assumed that the order status always reflects the status as it is known to the sending application at the time that the message is sent. Only the filler can originate the value of this field.

[segment-comment]
Although {{v2-table:0038:Order status}} contains many of the same values contained in {{v2-table:0119:Order control codes and their meaning}}, its purpose is different. Order status may typically be used in a message with an ORC-1-order control value of SR or SC to report the status of the order on request or to any interested party at any time.
