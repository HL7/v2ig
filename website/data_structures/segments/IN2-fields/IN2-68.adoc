== IN2.68 - Stoploss Limit Flag (ID)

[segment-definition]
This field indicates if the patient has reached the stoploss limit established in the Contract Master. Refer to {{v2-table:0136:Yes/no Indicator}} in Chapter 2C, Code Tables, for valid values.

[segment-comment]
Y the patient has reached the stoploss limit

N the patient has not reached the stoploss limit
