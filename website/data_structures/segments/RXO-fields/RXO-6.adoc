== RXO.6 - Provider's Pharmacy/Treatment Instructions (CWE)

[segment-definition]
This field identifies the ordering provider's instructions to the pharmacy or the non-pharmacy treatment provider (e.g., respiratory therapy). If coded, a user-defined table must be used. If transmitted as a free text field, place a null in the first component and the text in the second, e.g., |^this is a free text treatment instruction|.Refer to Table 0751 - Provider's Pharmacy/Treatment Instructions in Chapter 2C for valid values.

[segment-comment]
If the prescription is transmitted as free text using RXO-6, then RXO-1, RXO-2, and RXO-4 may be blank and the first subcomponent of RXO-6 must be blank. Otherwise, RXO-1, RXO-2 and RXO-4 are mandatory.
