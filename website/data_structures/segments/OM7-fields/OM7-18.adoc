== OM7.18 - Consent Waiting Period Units (CWE)

[segment-definition]
This field specifies the unit of time for _OM7-17 - Consent Waiting Period Quantity_. Refer to {{v2-table:0414:Units of time}} in Chapter 2C, Code Tables, for suggested values.

[segment-comment]
*Note:* If Consent Waiting Period Quantity is specified, then Consent Waiting Period Unit is required.
