== RXV.9 - PCA Dose Amount Units (CWE)

[segment-definition]
This field indicates the amount units associated with the PCA dose amount. The preferred coding system is MDC; UCUM are also acceptable. Refer to Table 0769 - PCA Dose Amount Units in Chapter 2C for valid values.

[segment-comment]
Examples:

263890^MDC_DIM_MILLI_G^MDC

mg^milligram^UCUM
