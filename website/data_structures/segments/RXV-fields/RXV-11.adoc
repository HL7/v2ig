== RXV.11 - PCA Dose Amount Volume Units (CWE)

[segment-definition]
This field indicates the amount units associated with the PCA dose volume. The preferred coding system is MDC; UCUM are also acceptable. Refer to Table 0770 - PCA Dose Amount Volume Units in Chapter 2C for valid values.

[segment-comment]
Examples:

263890^MDC_DIM_MILLI_G^MDC

mg^milligram^UCUM
