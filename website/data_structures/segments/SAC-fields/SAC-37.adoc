== SAC.37 - SAC-37 (CWE)

[segment-definition]
This field is the unit's identifier that is being used to describe the Icterus Index of the specimen. It is recommended to use mMol/L of bilirubin. Refer to Table 0781 - Icterus Index Units in Chapter 2C for valid values.

[segment-comment]
If this field is null, the recommended value is assumed.
