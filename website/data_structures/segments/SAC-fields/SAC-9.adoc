== SAC.9 - SAC-9 (CWE)

[segment-definition]
This field identifies the type of the carrier (see section _13.1.3_, "_Glossary_"). Refer to {{v2-table:0378:Carrier Type}} in Chapter 2C, Code Tables, for suggested values. Because the geometry can be different, the carrier type should, if possible, express the number of positions in the carrier.

[segment-comment]
The definition assumes hierarchical nesting using the following phrases: container is located in a carrier; carrier is located in a tray.
