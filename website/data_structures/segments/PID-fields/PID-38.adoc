== PID.38 - Production Class Code (CWE)

[segment-definition]
This field contains the code and/or text indicating the primary use for which the living subject was bred or grown. Refer to {{v2-table:0429:Production Class Code}} in Chapter 2C, Code Tables, for suggested values. For example:
