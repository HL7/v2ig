== CDM.6 - Exploding Charges (CWE)

[segment-definition]
This field contains the repeating occurrences for a list of other CDM entry charge codes identifying the other charges which should be generated from this CDM entry. Refer to {{v2-table:0132:Transaction Code}} in Chapter 2C, Code Tables, for suggested values. If non-null, posting a charge to this CDM entry should result in posting the charges identified here. These are sometimes called "linked items."

[segment-comment]
In the case of "chained" charges where the "lead" charge must be included in the exploded charges, the "lead" charge should be included in the list of exploding charges. If the price of this parent charge is included in the message, then it overrides the sum of the exploded charges prices.
