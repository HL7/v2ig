== CDM.7 - Procedure Code (CNE)

[segment-definition]
This field contains the procedure code for procedure, if any, associated with this charge description. Repeating field allows for different procedure coding systems such as CPT4, ICD9. Coded entry made up of code plus coding schema. Refer to {{v2-table:0088:Procedure Code}} in Chapter 2C, Code Tables, for suggested values.
