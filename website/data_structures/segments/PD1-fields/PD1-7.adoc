== PD1.7 - Living Will Code (CWE)

[segment-definition]
This field indicates whether or not the patient has a living will and, if so, whether a copy of the living will is on file at the healthcare facility. If the patient does not have a living will, the value of this field indicates whether the patient was provided information on living wills. Refer to {{v2-table:0315:Living Will Code}} in Chapter 2C, Code Tables, for suggested values. See also _PV2-43 - Living Will Code_.
