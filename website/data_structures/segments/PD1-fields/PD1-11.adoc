== PD1.11 - Publicity Code (CWE)

[segment-definition]
This field contains a user-defined code indicating what level of publicity is allowed (e.g., No Publicity, Family Only) for the patient. Refer to {{v2-table:0215:Publicity Code}} in Chapter 2C, Code Tables, for suggested values. Refer to _PV2-21 - Visit Publicity Code_ for visit level code.
