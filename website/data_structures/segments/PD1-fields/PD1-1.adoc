== PD1.1 - Living Dependency (CWE)

[segment-definition]
This field identifies specific living conditions (e.g., spouse dependent on patient, walk-up) that are relevant to an evaluation of the patient's healthcare needs. This information can be used for discharge planning. This field repeats because, for example, "spouse dependent" and "medical supervision required" can apply at the same time. Refer to {{v2-table:0223:Living Dependency}} in Chapter 2C, Code Tables, for suggested values.
