== PV1.14 - Admit Source (CWE)

[segment-definition]
This field indicates from where the patient was admitted. Refer to {{v2-table:0023:Admit Source}} in Chapter 2C, Code Tables, for suggested values. In the US, this field should use the Official Uniform Billing (UB) 04 2008 numeric codes found on form locator 15. Refer to _External Table UB04FL15 Source of Origin_ for valid values. The UB has redefined the Admission Source as the Point of Origin for Admission or Visit. The new UB definition is the code indicating the Point of Origin for this Admission or Visit

[segment-comment]
For Example:

US: |1^Physician Referral^UB04FL15^^^^2008, v 2.0|

To report that information is not available:

US: |9^Information not available^ UB04FL15^^^^2008, v 2.0|
