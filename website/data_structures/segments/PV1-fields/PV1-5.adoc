== PV1.5 - Preadmit Number (CX)

[segment-definition]
This field uniquely identifies the patient's pre‑admit account. Some systems will continue to use the pre‑admit number as the billing number after the patient has been admitted. For backward compatibility, a ST data type can be sent; however, HL7 recommends use of the CX data type, like the account number, for new implementations. The assigning authority and identifier type codes are strongly recommended for all CX data types.
