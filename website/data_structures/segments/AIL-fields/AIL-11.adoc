== AIL.11 - Allow Substitution Code (CWE)

[segment-definition]
This field contains a code indicating whether the identified location can be replaced with an equivalent substitute location by the filler application. Refer to {{v2-table:0279:Allow Substitution Codes}} in Chapter 2C, Code Tables, for suggested values.

[segment-comment]
This field is conditionally required. It is required for all request messages. It is optional for all unsolicited transactions, and for all query messages.
