== IN3.20 - Pre-Certification Requirement (ICD)

[segment-definition]
This field indicates whether pre-certification is required for particular patient types, and the time window for obtaining the certification. The following components of this field are defined as follows:

[segment-comment]
• pre-certification required refers to {{v2-table:0136:Yes/no Indicator}} in Chapter 2C, Code Tables, for valid values.

Y pre-certification required

N no pre-certification required

• pre-certification window is the date/time by which the pre-certification must be obtained.
