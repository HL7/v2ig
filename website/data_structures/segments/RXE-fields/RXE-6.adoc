== RXE.6 - Give Dosage Form (CWE)

[segment-definition]
The dosage form indicates the manner in which the medication or treatment is aggregated for dispensing, e.g., tablets, capsules, suppositories. In some cases, this information is implied by the give code in _RXE-2-Give Code_. Use the _RXE-6-Give Dosage Form_ when the give code does not specify the dosage form. Refer to Table 0716 - Give Dosage Form in Chapter 2C for valid values.
