== RXE.31 - Supplementary Code (CWE)

[segment-definition]
This field accommodates the identification of any codes that might be associated with the pharmaceutical substance. Common codes include: the Generic Product Identifier (GPI), Generic Code Number_Sequence Number (GCN_SEQNO), National Drug Code (NDC). Refer to Table 0727 - Supplementary Code in Chapter 2C for valid values.
