== PRA.11 - Government Reimbursement Billing Eligibility (CWE)

[segment-definition]
This field contains a code that indicates the agency that the practitioner is authorized to bill for medical services. Refer to {{v2-table:0401:Government Reimbursement Program}} in Chapter 2C, Code Tables,for valid values. This table contains the associated code values for medicaid and medical. This is a repeating field.
