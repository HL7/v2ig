== SCD.32 - Procedure Code (CNE)

[segment-definition]
The unique identifier indicating the type of procedure performed on the patient with the supplies being sterilized.

[segment-comment]
Refer to {{v2-table:0088:Procedure Code}} in Chapter 2C, Code Tables, for suggested values.

As of v2.6, the known applicable external coding systems include those in the referenced table. If the code set you are using is in this table, then you must use that designation.
