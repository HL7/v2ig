== MRG.1 - Prior Patient Identifier List (CX)

[segment-definition]
This field contains the prior patient identifier list. This field contains a list of potential "old" numbers to match. Only one old number can be merged with one new number in a transaction. Refer to {{v2-table:0061:Check Digit Scheme}} for valid values. The assigning authority and identifier type code are strongly recommended for all CX data types.
