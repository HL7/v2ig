== OBR.44 - Procedure Code (CNE)

[segment-definition]
This field contains a unique identifier assigned to the procedure, if any, associated with the charge. Refer to {{v2-table:0088:Procedure code}}_ in Chapter 2C, Code Tables, for suggested values__.__ This field is a coded data type for compatibility with clinical and ancillary systems.

[segment-comment]
As of version 2.6, applicable external coding systems include those in the referenced table. If the code set used is in the referenced table, then the coding scheme designation in the table shall be used.
