== STF.2 - Staff Identifier List (CX)

[segment-definition]
This field contains the list of identifiers (one or more) used by the institution to identify this person. Refer to {{v2-table:0061:Check Digit Scheme}} (component 2), {{v2-table:0203:Identifier Type}} (component 5) and {{v2-table:0363:Assigning Authority}} (component 4) for valid values (see Chapter 2A). If a Staff Member has institution identifiers that are valid only in specific organization units, that Staff identifier should be recorded in the _ORG Practitioner Organization Unit_ segment.
