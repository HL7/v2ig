== INV.1 - INV-1 (CWE)

[segment-definition]
Unique identifier for the substance that is in inventory. This is a manufacturer-specific identifier. Refer to {{v2-table:0451:Substance Identifier}} in Chapter 2C, Code Tables, for suggested values.

[segment-comment]
This field is conditional. It is optional in queries INR^U14 and required in all other messages. Absence of this field in a query means that the result shall be not filtered on inventory item ID.
