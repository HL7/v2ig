== DG1.26 - Present On Admission (POA) Indicator (CWE)

[segment-definition]
This field contains the present on admission indicator for this particular diagnosis. US reimbursement formulas for some states and Medicare have mandated that each diagnosis code be flagged as to whether it was present on admission or not. Refer to {{v2-table:0895:Present On Admission (POA) Indicator}} in Chapter 2C, Code Tables, for suggested values.
