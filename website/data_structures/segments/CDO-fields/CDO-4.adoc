== CDO.4 - Cumulative Dosage Limit Time Interval (CQ)

[segment-definition]
The Cumulative Dosage Time Interval indicates the interval over which the cumulative dosage in _CDO-3 Cumulative Dosage Limit_ is measured. If this field is not valued, the Cumulative Dosage Limit applies to the scope of the order.

[segment-comment]
Refer to {{v2-table:924:Cumulative Dosage Limit UoM}} in Chapter 2C, Code Tables, for a list of suggested values, including applicable values from UCUM.
