== ITM.28 - Procedure Code Modifier (CNE)

[segment-definition]
This field contains the procedure code modifier to the procedure code reported in _ITM-27, Procedure Code_, when applicable. Procedure code modifiers are defined by USA regulatory agencies such as CMS and the AMA. Multiple modifiers may be reported. Refer to {{v2-table:0340:Procedure Code Modifier}} in Chapter 2C, Code Tables, for suggested values.
