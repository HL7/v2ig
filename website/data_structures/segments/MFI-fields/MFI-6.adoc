== MFI.6 - Response Level Code (ID)

[segment-definition]
These codes specify the application response level defined for a given Master File Message at the MFE segment level as defined in {{v2-table:0179:Response Level}} in Chapter 2C, Code Tables. Required for MFN-Master File Notification message. Specifies additional detail (beyond _MSH-15 - Accept Acknowledgment Type_ and _MSH-16 - Application Acknowledgment Type_) for application-level acknowledgment paradigms for Master Files transactions. _MSH-15 - Accept Acknowledgment Type_ and _MSH-16 - Application Acknowledgment Type_ operate as defined in Chapter 2.
