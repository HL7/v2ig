== MFI.2 - Master File Application Identifier (HD)

[segment-definition]
This field contains an optional code of up to 180 characters which (if applicable) uniquely identifies the application responsible for maintaining this file at a particular site. A group of intercommunicating applications may use more than a single instance of a master file of certain type (e.g., charge master or physician master). The particular instance of the file is identified by this field. Refer to {{v2-table:0361:Applications}}.
