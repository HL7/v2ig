== MSH.3 - Sending Application (HD)

[segment-definition]
This field uniquely identifies the sending application among all other applications within the network enterprise. The network enterprise consists of all those applications that participate in the exchange of HL7 messages within the enterprise. Entirely site-defined. E:\\V2\\v2.9 final Nov from Frank\\V29_CH02C_Tables.docx#HL70361[_User-defined Table 0361- Application_] in Chapter 2C, Code Tables, is used as the user-defined table of values for the first component.
