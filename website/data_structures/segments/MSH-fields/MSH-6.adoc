== MSH.6 - Receiving Facility (HD)

[segment-definition]
This field identifies the receiving application among multiple identical instances of the application running on behalf of different organizations. {{v2-table:0362:Facility}} in Chapter 2C, Code Tables, is used as the HL7 identifier for the user-defined table of values for the first component. Entirely site‑defined.

[segment-comment]
*Note*: By site agreement, implementers MAY continue to use {{v2-table:0300:Namespace ID}} in Chapter 2C, Code Tables, for the first component.
