== AIS.12 - Filler Supplemental Service Information (CWE)

[segment-definition]
This field contains supplemental service and/or logistical information sent from the filler system to the placer system for the procedure code reported in field AIS-3. This field will be used to report scheduling information details that is not available in other, specific fields in the AIS segment. Typically it will reflect the same information as was sent to the filler system in _AIS-11-Placer Supplemental_ information unless the scheduling was modified in which case the filler system will report what was actually performed using this field. Multiple supplemental service information elements may be reported. Refer to {{v2-table:0411:Supplemental service information values}} in Chapter 2C, Code Tables, for valid values..
