== DRG.14 - PCCL Value Code (CWE)

[segment-definition]
This field indicates the PCCL (Patient Clinical Complexity Level) value for the calculated DRG as a single value. This value is calculated based on all individual CCL values calculated so far in relation to the basic DRG. Refer to {{v2-table:0728:CCL Value}} in Chapter 2C, Code Tables, for suggested values.
