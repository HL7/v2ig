== MFA.1 - Record-Level Event Code (ID)

[segment-definition]
This field defines record-level event for the master file record identified by the MFI segment and the primary key in this segment. Refer to {{v2-table:0180:Record-level Event Code}} in Chapter 2C, Code Tables, for valid values.

[segment-comment]
*Note:* If the _MFI-3 - File-level event code_ is "REP" (replace file), then each MFA segment must have an _MFA-1 - Record-level event code_ of "MAD" (add record to master file).
