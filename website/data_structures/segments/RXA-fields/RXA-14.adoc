== RXA.14 - Administered Strength Units (CWE)

[segment-definition]
Use when _RXA-5-Administered Code_ does not specify the strength. This is the unit of the strength, used in combination with _RXA-13-Administered Strength_. Refer to Table 0692 - Administered Strength Units in Chapter 2C for valid values.

[segment-comment]
*Note*: These units can be a "compound quantity;" i.e., the units may express a quantity per unit of time. For example, micrograms per hour (micg/h) is an acceptable value. These compound units are contained in the ISO+ table. See Chapter 7 for full definition of ISO+ units.
