== BPO.13 - BP Indication for Use (CWE)

[segment-definition]
This is a coded optional field. The value indicates the reason that the blood product was ordered. This information is helpful for prospective review or retrospective studies of blood product ordering practices of the ordering provider by the Quality Assurance Department and/or Transfusion Committee. Refer to {{v2-table:0509:Indication for Use}} in Chapter 2C, Code Tables, for suggested values.
