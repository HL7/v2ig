== PTH.7 - Mood Code (CNE)

[segment-definition]
This field indicates the Mood of the Pathway. It allows expression of the context of the Pathway.

[segment-comment]
*Note:* As Mood Code changes the meaning of the segment it must only be used in new messages as of v2.6.

Refer to {{v2-table:0725:Mood Codes}}_ in Chapter 2C, Code Tables, for allowed values.
