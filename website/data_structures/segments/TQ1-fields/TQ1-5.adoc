== TQ1.5 - Relative Time and Units (CQ)

[segment-definition]
This field is used to define the interval between schedules for service request or bottle records. If this field contains a value, it overrides any value in the explicit time interval field. The units component of the CQ data type is constrained to units of time.

[segment-comment]
Examples:

TQ1|1|1|Q1H||60^min&&ANS+ - Q1H is defined with an interval between services of 60 minutes

TQ1|1|1|Q6H||6^hr&&ANS+ - Q6H is defined with an interval between services of 6 hours

TQ1|1|1|QD||1^d&&ANS+ - QD is defined with an interval between services of 1 day
