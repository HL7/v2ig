== OH2.4 - Occupation (CWE)

[segment-definition]
This field contains a code defining the occupation in which an individual is engaged in for the job recorded in this segment, regardless of duration. Refer to {{v2-table:0958:Occupation Code}} in Chapter 2C, Code Tables, for suggested values. In the US, this field should use the Occupation CDC Census 2010 codes published at https://phinvads.cdc.gov/vads/ViewValueSet.action?oid=2.16.840.1.114222.4.11.7186
