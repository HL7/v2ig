== OH2.5 - Industry (CWE)

[segment-definition]
This field contains a code defining the industry in which an individual is engaged for the job recorded in this segment, regardless of duration. Refer to {{v2-table:0955:Industry Code}} in Chapter 2C, Code Tables, for suggested values. In the US, this field should use the Industry CDC Census 2010 codes published at https://phinvads.cdc.gov/vads/ViewValueSet.action?oid=2.16.840.1.114222.4.11.7187
