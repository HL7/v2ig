== OH2.1 - Set ID (SI)

[segment-definition]
This field contains the number that identifies the OH2 segment instances in message structures where the OH2 segment repeats. For the first occurrence of the segment, the sequence number shall be one, for the second occurrence, the sequence number shall be two, etc. If the subject of the OH2 segment changes (e.g. Patient vs Next of Kin) the Set ID sequence will be reset.
