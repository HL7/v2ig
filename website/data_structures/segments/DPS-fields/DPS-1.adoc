== DPS.1 - Diagnosis Code - MCP (CWE)

[segment-definition]
DPS-1 contains the diagnosis code assigned to this diagnosis. Refer to E:\\V2\\v2.9 final Nov from Frank\\V29_CH02C_Tables.docx#HL70051[User-defined Table 0051 - Diagnosis Code] for suggested values. This field is a CWE data type for compatibility with clinical and ancillary systems. Either _DPS-1.1-Identifier_ or _DPS-1.2-Text_ is required. When a code is used in _DPS-1.1-Identifier_, a coding system is required in _DPS-1.3-Name of Coding System_.

[segment-comment]
Names of various diagnosis coding systems are listed in Chapter 2, Section 2.16.4, “Coding system table.”
