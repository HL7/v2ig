== SPM.8 - Specimen Source Site (CWE)

[segment-definition]
specifies the source from which the specimen was obtained. For example, in the case where a liver biopsy is obtained via a percutaneous needle, the source would be 'liver'. Refer to Table 0784 - Specimen Source Site in Chapter 2C for valid values.Any nationally recognized coding system might be used for this field including SNOMED; alternatively the {{v2-table:0163:Body Site}} may be used. Veterinary medicine may choose the tables supported for the components of this field as decided by their industry.
