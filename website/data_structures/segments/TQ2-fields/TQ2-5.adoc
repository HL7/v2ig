== TQ2.5 - Related Placer Group Number (EI)

[segment-definition]
The placer group numbers of the service request(s) to which this TQ2 segment links the current service request. This field should be populated with the appropriate "Placer group number" from the current service request. For orders, the Placer Group Number from ORC-4 is the appropriate "Placer group number". Repeats of this field indicate that the current service request is related to multiple groups of service requests.

[segment-comment]
Conditional Rule: At least one of TQ2-3, TQ2-4, TQ2-5 must contain a value.
