== TQ2.3 - Related Placer Number (EI)

[segment-definition]
The placer numbers of the service request(s) to which this TQ2 segment links the current service request. This field should be populated with the appropriate "Placer number" from the current service request. For orders, the Placer Order Number from ORC-2 is the appropriate "Placer number". Repeats of this field indicate the current service request is related to multiple service requests.

[segment-comment]
Conditional Rule: At least one of TQ2-3, TQ2-4, TQ2-5 must contain a value.
