== NK1.27 - Nationality (CWE)

[segment-definition]
This field identifies the nation or national group to which the next of kin/associated party belongs. This information may be different than the person's citizenship in countries in which multiple nationalities are recognized (e.g., Spain: Basque, Catalan, etc.). Refer to {{v2-table:0212:Nationality}} in Chapter 2C, Code Tables, for suggested values.
