== XCN.13 - Identifier Type Code (ID)

[datatype-definition]
A code corresponding to the type of identifier. In some cases, this code may be used as a qualifier to the <assigning authority> component. Refer to {{v2-table:0203:Identifier Type}} for suggested values.

As of v 2.7, _XCN.13_ is conditional. It is required if _XCN.1_ is populated.
