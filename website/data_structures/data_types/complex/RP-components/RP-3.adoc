== RP.3 - Type of Data (ID)

[datatype-definition]
An ID data type that declares the general type of data. Refer to {{v2-table:0834:MIME Types}} for valid values. Table 0834 comprises the values of MIME media types established in accordance with RFC 2046 (http://ietf.org/rfc/rfc2046.txt) and registered with the Internet Assigned Numbers Authority (http://www.iana.org/numbers.html).

[NOTE]
The MIME media type values are case-insensitive, in accordance with RFC 2045.

[NOTE]
{{v2-table:0191:Type of Referenced Data}} is retained for backward compatibility as of v 2.6.*
