== NDL.9 - Patient Location Type (IS)

[datatype-definition]
Location type is the categorization of the location defined by facility, building, floor, point of care, room or bed. Although not a required field, when used, it may be the only populated field. Usually includes values such as nursing unit, department, clinic, SNF, physician’s office. Refer to {{v2-table:0305:Person Location Type}} for suggested values.
