# V2+ MVP Test IG

This is a deliberately minimal Implementation Guide used to exercise the
HL7 auto-ig-builder against trivial inputs. It exists on the `mvp-test`
branch of [HL7/v2ig](https://github.com/HL7/v2ig) to help diagnose whether
the parent IG's build failures are content-specific or pipeline-wide.

If this page is rendering, auto-ig-builder is functioning end-to-end for
this branch and the terminology-server connection succeeded during init.
